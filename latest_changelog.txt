Load conditions for tags don't seem to be working (randomium blacklist, projecte ignore missing emc). This means the tags will be loaded even if the respective mod isn't. This won't cause any issues other than unnecessary bloat in the tags. (check this with new mod jar)

# Info about next version

I might start learning java so I can rewrite the mod to use datagen soon. This will make adding compat much easier in the future, since it is starting to get a bit tedious, but it would likely take a while to set up.

Once the mod is set up, I would be able to add support for many more mods, since adding them would be much faster, since I would only have to add a single mod ID and material name in the right place, like "phantasm" and "pream" and it would do all the wood recipes for End's Phantasm.

However, I might deprecate some mods/features to refine the scope of this mod and improve compatability. Most notably ProjectE custom conversions. I will still add world transmutations, but [ProjectE Integration](https://modrinth.com/mod/projecte-integration) does a better job at handling the EMC values.

Ore processing will also become a lower priority since JAOPCA does a better job at handling that. I will still add some recipes (with the appropriate load conditions from the beginning) and a few that aren't handled by JAOPCA like drilling for ores with Modern Industrialization, which will be a bit more of a stylized tweak.

Basically, don't expect any **major** updates soon. There won't be an ETA. It could be a month or it could be a year. The mod is currently set up to work on any Minecraft version above 1.21, so unless there are any breaking changes, all I need to do is update the version on Modrinth for it to show for new versions, but it'll work anyway if you install the mod manually.

There will still be a few smaller changes and additions to the current mod whilst the rewrite is in development, but not any big new features.

I will likely make the mod using something like [Architectury](modrinth.com/mod/architectury-api) so it's available natively on forge, neoforge, fabric and quilt. There _might_ also be versions below 1.21.

I'll probably also change the license to MIT for the rewrite.
